# 项目简介

本项目为基于深度学习的血红细胞检测计数系统，采用YOLOv1算法和Python编程语言实现。系统包含用户界面，便于操作。主要功能为自动检测血红细胞并计数，技术特点包括深度学习模型训练、图像识别及处理。该系统不涉及售卖和交易，仅供学习和研究使用，禁止用于商业目的。含数据集，可供用户进行模型训练和测试。


## 核心功能
- 基于YOLOv1算法的血红细胞自动检测与计数
- 提供用户界面，便于操作使用
- 包含完整的数据集，支持模型训练和测试
- 实现图像识别及处理功能

## 适用场景
- 医学研究领域中的血红细胞分析
- 医学图像处理相关学习
- 深度学习模型训练实践
- 医学计数自动化研究

## 技术特点
- 采用Python编程语言实现
- 基于深度学习模型进行训练
- 集成YOLOv1算法进行目标检测
- 提供完整的UI界面交互
- 包含专用数据集支持研究工作

该项目仅用于学习和研究目的，禁止商业使用。
